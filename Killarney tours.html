How to Fix It and Install Software:

Option 1: Disable Secure Boot (Recommended for Installing Software)

  1. Restart your computer and enter the BIOS/UEFI menu.
    • Press F2, F12, DEL, or ESC (varies by manufacturer) during startup.

  3. Disable Secure Boot and save changes.
  4. Restart the system and try booting or installing software again.

Option 2: Enable Compatibility Mode (CSM)

If disabling Secure Boot doesn’t work, enable Legacy/CSM boot mode in the BIOS. This allows booting from older devices.

Option 3: Use a Signed Bootloader (For Installing OS)

  • If you are installing Linux, use a version that supports Secure Boot (Ubuntu, Fedora, etc.).
  • If you are booting from a USB drive, create a UEFI-compatible bootable USB using Rufus (select "GPT + UEFI" when formatting).

Як скопіювати код index.html з вкладки "Tour Guide Website"

  1. Переконайся, що вкладка "Tour Guide Website" відкрита (це код твого сайту).
  2. Виділи весь код:
    • Натисни Ctrl + A (Windows) або Cmd + A (Mac) – це виділить увесь код.
  3. Скопіюй код:
    • Натисни Ctrl + C (Windows) або Cmd + C (Mac).
  4. Відкрий "Блокнот" або текстовий редактор:
    • У Windows – натисни Win + R, введи notepad, натисни Enter.
    • У Mac – відкрий "TextEdit".
    • Можна використовувати VS Code, Sublime Text або Notepad++.
  5. Встав код у файл:
    • Натисни Ctrl + V (Windows) або Cmd + V (Mac).
  6. Збережи файл як index.html:
    • Натисни Файл → Зберегти як....
    • У полі "Ім'я файлу" введи: index.html.
    • У полі "Тип файлу" вибери Усі файли (*.*).
    • Натисни Зберегти.
